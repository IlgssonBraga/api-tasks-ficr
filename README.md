## Description

A REST api made with Express.js and Sequelize to tasks controller.
## Installation

```bash
$ git clone https://github.com/IlgssonBraga/api-tasks-ficr.git
```

## Running the app

```bash
$ npm install

```

```bash
$ npm run db:migrate 

```

```bash
$ npm run start:dev

```
